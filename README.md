- [x] Create account

- [x] Edit my profile
- [x] See user profile
- [x] See my profile
- [x] See the full post
- [x] Upload a post
- [x] Edit / Delete a post
- [x] See the feed
- [x] See message rooms
- [x] Send direct message 
- [x] See a specific message room
- [x] Receive message (real-time)
